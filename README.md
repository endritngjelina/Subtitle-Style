# 🎬 Subtitle Style

A lightweight, browser-based tool for designing and previewing custom subtitle styles. Built entirely with vanilla HTML, CSS, and JavaScript — no frameworks or dependencies required.

---

## 🚀 Features

- Live preview of subtitle text styles
- Customize fonts, colors, sizes, and text effects
- Adjust subtitle backgrounds, borders, and opacity
- Experiment with positioning and alignment
- Explore 16+ design iterations (v0–v15)
- Fully in-browser — no installation or server needed

---

## 🛠️ Usage

1. Clone or download the repository:
   ```bash
   git clone https://github.com/endritngjelina/Subtitle-Style.git
   ```
2. Open any `.html` file directly in your browser — no setup required.
3. Recommended: Open **Subtitle Style 15.html** for the most complete version.

---

## 💡 Use Cases

- Designing subtitle overlays for social media videos
- Prototyping caption styles for web video players
- Creating accessible, high-contrast subtitle themes
- Exploring CSS typography and text effect combinations
